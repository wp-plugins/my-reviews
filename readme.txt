Add, edit, and delete reviews. Show reviews to your readers using a single review view, archive view, and using widgets. Feature specific reviews. Optionally connect the plugin to your GuestRetain account to automatically sync reviews.

== Description ==

Add, edit, and delete reviews through a stylish user interface. The plugin creates a review archive using your existing theme structure. It also creates a single review view which you cam tweet, Facebook share, Google+ share, etc.

= Features: =

* User friendly interface to create, edit, and delete reviews
* Featured and latest reviews widget
* Review archive
* Single review view
* Optional: sync reviews from [GuestRetain](http://guestretain.com)


== Installation ==

1. Upload plugin folder
1. Activate
1. Done!
